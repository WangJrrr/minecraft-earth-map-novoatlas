# 安装与服务器部署

## 新世界

1. 安装 Minecraft 1.21.1 与 NeoForge。
2. 客户端和服务端同时安装 Create 及其依赖。
3. 客户端和服务端同时安装本仓库 `dist` 中的修改版 NovoAtlas。
4. 从 Releases 下载一个地图 ZIP，并解压为普通数据包文件夹。
5. 将数据包放入新世界的 `datapacks`，再创建或启动世界。

四个地图包分别对应两个比例和两种富矿规则。不要同时加载 Original 与 Top100，也不要同时加载 1:200 与 1:400。

## 已存在的世界

数据包只控制新区块。替换数据包后，已经生成的区块不会自动改变。若必须更新某一区域，需要删除对应区块或创建新世界；操作正式服务器存档前必须备份。

## Chunky

地图可以边探索边生成，不要求事先完成全球预生成。Chunky 适合提前生成出生点、交通走廊和主要活动区域。全球完整预生成会产生极大的存档，并消耗大量 CPU、磁盘写入和时间。

## 客户端与服务端

NovoAtlas 和 Create 均应在客户端与服务端安装。数据包本身放在服务端世界目录；单人游戏则放在对应存档的 `datapacks`。
