# 数据来源与署名

[English](../en-US/DATA_SOURCES.md) | [简体中文](DATA_SOURCES_zh-CN.md)

地图生成管线使用或参考以下数据：

- [ESA WorldCover 2021](https://esa-worldcover.org/en)：全球土地覆盖分类。
- [WorldClim 2.1](https://www.worldclim.org/data/worldclim21.html)：全球温度、降水等气候变量。
- [HydroRIVERS](https://www.hydrosheds.org/products/hydrorivers)：全球河流网络、流量和汇水属性。
- [Natural Earth](https://www.naturalearthdata.com/)：湖泊与辅助全球矢量图层。
- 构建管线缓存的全球高程瓦片。
- [USGS Mineral Resources Data System (MRDS)](https://www.usgs.gov/publications/mineral-resources-data-system-mrds)：矿产点、矿种和历史矿床元数据。
- [Global Energy Monitor Global Coal Mine Tracker](https://www.gem.wiki/Global_Coal_Mine_Tracker)：主要煤矿补充锚点。
- 项目人工整理的重要矿区锚点。

本仓库不重新分发第三方原始数据或大型缓存。复现者应从各数据提供方获取数据，并遵守对应许可、署名和再分发条款。

MRDS 是历史资源清单，不一定反映矿山当前状态、所有权、产量或储量。项目富矿排名属于游戏化转换，不应解释为现实经济或地质评估。
